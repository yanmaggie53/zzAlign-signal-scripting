# CSV Auto-Mover for Dashboard Integration

This system automatically moves CSV files generated by your conversion scripts to your Cursor dashboard's data directory.

## Setup Instructions

1. **Configure the Dashboard Path**:
   - Open `config.py`
   - Update `DASHBOARD_PATH` to point to your Cursor dashboard project directory
   - Example: `DASHBOARD_PATH = r"C:\Users\YourName\Cursor\Projects\SleepDashboard"`

2. **Ensure Dashboard Data Directory**:
   - Your dashboard should have a `data/` folder where CSV files will be placed
   - The auto-mover will create this folder if it doesn't exist

3. **Start the Auto-Mover**:
   - Double-click `start_auto_mover.bat` or run `python csv_auto_mover.py`
   - The system will monitor all conversion folders for new CSV files
   - When a CSV is created, it will be automatically copied to your dashboard

## How It Works

- Monitors: `Sleep Signal Conversion Scripting/`, `Sleep Stats Conversion Scripting/`, `Pump Signal Conversion Scripting/`
- Automatically copies any `.csv` files to: `DASHBOARD_PATH/data/`
- Runs continuously until stopped (Ctrl+C)

## Integration with Your Workflow

1. Run your conversion scripts as usual
2. CSV files are automatically moved to dashboard
3. Your dashboard can watch the `data/` folder or reload when files change

## Troubleshooting

- Check that `DASHBOARD_PATH` in `config.py` is correct
- Ensure your dashboard project exists at that location
- Make sure you have write permissions to the dashboard directory
